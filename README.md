1. What is the difference between var, let, and const?
   
Answer: var is function scoped and can be redeclared.

	let is block scoped and can be reassigned but not redeclared in the same scope.

	const is also block scoped but cannot be reassigned after declaration.
	
2. What is the spread operator (...)?

Answer: The spread operator (...) is used to expand elements of an array or object.

3. What is the difference between map(), filter(), and forEach()?

Answer: map() → creates a new array.

	filter() → returns elements that match condition.

	forEach() → just loops all elements, returns nothing.

4. What is an arrow function?

Answer: Arrow functions are shorter syntax for writing functions.

5. What are template literals?

Answer: Template literals allow string interpolation using backticks (``). They allow us to easily insert variables inside a string.
